title: "Sensitive Data"
---

You might want an overview off the PII data your endpoints process. On the Sensitive Data page you can get an overview of all the PII data your organization processes across all endpoints and hosts. This data can be filtered by:

* Host
* Risk Score
* Request Location
![3808](/images/9e5a031-Screenshot_2022-11-26_at_6.29.48_PM.png "Screenshot 2022-11-26 at 6.29.48 PM.png") 

With this dashboard you can answer questions like:

* Which endpoints are returning high risk PII data in the response body?
* How many endpoints are returning sensitive data?
* Which hosts have the most PII data?

We would love contributions to Metlo's repo. Here's what you need to know:

### Reporting bugs or issues

You can create a new issue on the Metlo repo
[here](https://github.com/metlo-labs/metlo/issues/new).

Before raising a new issue, please search within existing ones to make sure
you're not creating a duplicate.

<Warning>
Security Issues

If your issue is related to security, please email us directly at
akshay@metlo.com instead of creating an issue.

</Warning>

### Submitting Code

Anyone can can contribute to Metlo! To get started, follow our local development
guide. Then, make your change and submit a pull request to the
[repo](https://github.com/metlo-labs/metlo). Our team will review the PR and
merge it as soon as possible!
